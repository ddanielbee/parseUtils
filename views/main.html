Angular + Parse Utils

This is a series of code snippets that come in handy when developing Web Apps using Angular as a frontend framework and Parse as Backend.

Parse doesn't have a propietary Angular library, so some workarounds need to be made for Parse to speak in Angular language, specifically for Angular promises to be taken into account and the Angular digest to go through.

I will update this guide as I keep working with Parse and Angular. Also, anyone who wishes to contribute is welcome to do so, visit the project on github
